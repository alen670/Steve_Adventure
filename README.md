## 本地运行

**前置要求:** Node.js

1. 安装依赖:
   ```bash
   npm install
   ```
2. 在 [.env.local](.env.local) 中设置 `GEMINI_API_KEY` 为你的 Gemini API key
3. 运行应用:
   ```bash
   npm run dev
   ```

## 技术栈

- React + TypeScript
- Vite
- Capacitor (支持Android)
- Java 21 LTS
